
# Rotaract Club of Coimbatore City

Welcome to the official repository for the Rotaract Club of Coimbatore City website. This platform serves as a digital hub for our non-profit organization, facilitating community engagement, project management, and volunteer coordination.

## Project Overview

The Rotaract Club of Coimbatore City is a prominent community service organization under Rotary International District 3206. Our mission is to empower youth and serve the community through sustainable initiatives. This website is designed to showcase our projects, upcoming events, and provide a seamless way for volunteers and donors to contribute to our cause.

## Key Features

* **Donation System**: A streamlined interface for donors to contribute to various causes, with options for custom amounts.
* **Event Management**: Comprehensive listing of upcoming and past events, complete with details and registration capabilities.
* **Volunteer Registration**: An easy-to-use form for individuals to sign up and join our volunteer network.
* **Blog & News**: A dedicated section to share updates, success stories, and news about our initiatives.
* **Responsive Design**: Fully optimized for a seamless experience across desktops, tablets, and mobile devices.
* **Project Showcase**: Detailed portfolios of our ongoing and completed projects in education, healthcare, and livelihood support.

## Technology Stack

This project is built using modern web technologies to ensure performance and maintainability:

* **HTML5**: For semantic structure and accessibility.
* **CSS3 & SCSS**: For modular and scalable styling.
* **JavaScript (ES6+)**: For interactive elements and dynamic content.
* **Bootstrap**: Utilized for the responsive grid system and pre-built components.
* **Swiper & Splide**: Implemented for touch-friendly sliders and carousels.
* **Animate.css**: For smooth and engaging entry animations.

## Folder Structure

* `assets/`: Contains all static assets including CSS, SCSS, JavaScript files, images, and fonts.
* `documentation/`: Project documentation and guides.
* `*.html`: Main HTML pages for the website (Home, About, Contact, etc.).
