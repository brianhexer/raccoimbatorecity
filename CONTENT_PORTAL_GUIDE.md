## Editor Features

### Rich Text Editor
The rich text editor (for descriptions, bios, etc.) includes:

- **Text Formatting:** Bold, Italic, Underline
- **Headings:** H2, H3, H4, H5, H6
- **Lists:** Bullet points, Numbered lists
- **Links:** Add hyperlinks
- **Quotes:** Block quotes
- **Code:** Code blocks
- **Alignment:** Left, center, right, justify
- **Media:** Insert images, videos, audio
- **Undo/Redo:** Revert changes

**Tips:**
- Use headings to structure content
- Use lists for easy reading
- Add links to relevant pages
- Use media to make content engaging

### Featured Image
Upload an image for:
- Pages & posts (appears at top)
- Team members (profile photo)
- Events (poster image)
- Projects (cover image)

**Best sizes:**
- Posts/Pages: 1200x800px
- Team Photos: 400x500px
- Event Posters: 1200x600px
- Project Images: 800x600px
